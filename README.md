# 請求書 一括作成＆メール下書きツール（GAS）

スプレッドシートに請求内容を入力するだけで、**請求書PDFを一括生成し、取引先ごとにGmailの下書きまで自動で用意する** Google Apps Script ツールです。

サーバー不要・無料の Google アカウントだけで動きます。プログラミングの知識がなくても、シートに入力してメニューを押すだけで使えます。

## こんな悩みを解決します

- 毎月の請求書を、Excelのファイルをコピーして1枚ずつ手で作っている
- 作った請求書を、取引先ごとにメールに添付して送るのが地味に時間がかかる
- 件数が増えると、送り忘れ・金額の転記ミスが怖い

**Before → After（試算）**

| | これまで | このツール |
|---|---|---|
| 1件あたりの作成・送付 | 約10分（コピー・編集・PDF化・添付） | 一括処理で数秒 |
| 月20件の場合 | 約3.3時間 | 入力後、ボタン1つ＋確認のみ |

> ※数値は「1件10分 × 20件」を前提にした試算です。実際の効果は運用により変わります。

---

## できること

- スプレッドシートの請求データから **請求書PDFを一括生成**
- 同じ請求書番号の行を **1枚の請求書に自動でまとめる**（複数明細に対応）
- 小計・消費税（10%）・合計を **自動計算**
- 取引先ごとに **Gmailの下書きを自動作成**（PDF添付・本文つき）
- 処理前に **入力チェック**（メール形式・空欄・数量0以下）。問題があれば一切処理せず中止
- 処理した行に **ステータスと処理日時を記録**（二重送付を防ぐ目印）

### 安全のための設計

既定は「**下書き作成**」です。いきなり送信せず、内容を目で確認してから手動で送れます。
慣れて問題なければ「送信（自動）」メニューでまとめて送信もできます（実行前に確認ダイアログあり）。

## 設計の考え方（要件 → 実装）

このツールは「動けばいい」ではなく、**実務で事故が起きない**ことを優先して設計しています。

| 満たすべき要件 | 実装での対応 |
|---|---|
| 高コストな処理の前に入力ミスを潰す | PDF生成・メール作成の **前に** 全件をバリデーションし、1件でも不正なら処理を始めない |
| 誤送信を防ぐ | 既定を「下書き」にし、送信は別メニュー＋確認ダイアログ |
| 二重送付を防ぐ | 処理済みの行にステータスと日時を記録 |
| 非エンジニアが導入できる | メニューから「初期セットアップ」でサンプル入りのシートを自動生成 |
| 自社情報の変更が容易 | 会社名・振込先などはコードではなく「設定」シートで管理 |

## 導入手順

1. Googleスプレッドシートを新規作成する
2. メニュー **拡張機能 → Apps Script** を開く
3. `src/Code.gs` の中身を貼り付けて保存する
4. （任意）`appsscript.json` の内容を合わせると権限が最小構成になります
5. スプレッドシートを再読み込みすると、上部に **「請求書ツール」メニュー** が表示されます
6. 「① 初期セットアップ（サンプル作成）」を実行 → 「設定」「請求データ」シートが作られます

## スプレッドシートの構成

### 「設定」シート（自社情報）

| 項目 | 例 |
|---|---|
| 自社名 | 株式会社サンプル |
| 住所 | 東京都〇〇区〇〇1-2-3 |
| 電話番号 | 03-0000-0000 |
| メールアドレス | info@example.com |
| 振込先 | 〇〇銀行 〇〇支店 普通 1234567 カ）サンプル |
| 支払期限（発行日から日数） | 30 |

### 「請求データ」シート（請求内容）

| 請求書番号 | 発行日 | 取引先名 | 宛先メール | 品目 | 数量 | 単価 | ステータス | 処理日時 |
|---|---|---|---|---|---|---|---|---|
| INV-001 | 2026-06-30 | 取引先A株式会社 | a@example.com | Webサイト保守（6月分） | 1 | 30000 | | |
| INV-001 | 2026-06-30 | 取引先A株式会社 | a@example.com | サーバー利用料 | 1 | 5000 | | |
| INV-002 | 2026-06-30 | 取引先B商店 | b@example.com | 記事執筆（5本） | 5 | 8000 | | |

> 同じ **請求書番号** の行は、自動的に1枚の請求書（複数明細）にまとまります。
> 「ステータス」「処理日時」はツールが自動で書き込みます。

## 使い方

1. 「設定」シートに自社情報を入力
2. 「請求データ」シートに請求内容を入力
3. メニュー **請求書ツール → ② 請求書を作成（Gmail下書き）** を実行
4. Gmailの下書きを開いて内容を確認し、問題なければ送信
   （まとめて自動送信したい場合は **③ 請求書を送信（自動）**）

## カスタマイズ例

このテンプレートをベースに、こんな拡張がよく求められます。

- 請求書フォーマットを自社のデザイン・ロゴに合わせる
- 適格請求書（インボイス制度）の登録番号・税率区分に対応する
- 入金消込・未入金の自動チェック、督促メールの自動化
- freee / マネーフォワード などの会計ソフトと連携する
- 請求書を Google Drive の指定フォルダへ自動保存・採番する
